# Capstone Project CS30 - Smash Bros Clone

# Needs to Have
- Physics engine (jumping, vectors, acceleration)
- Player class (movement, stats, state)
- Attack class (angle, damage)
- Stage class (respawn points, size)
- Finite state machine (idle, running, airborne, jumpsquat, hitstun, starting, active, ending)
- Knockback calculator (percentage and weight based launch angles)
- Collision detection (AABB collision, rectangle hitboxes and hurtboxes)

# Nice to Have
- Animations and sprites
- Frame data
- Advanced stats
- Built-in game mechanics (shield, roll, airdodge, spotdodge, tech, getup attack, footstool)
- Ledge mechanics (ledge grab, ledge invincibility, ledge options)
- All smash character attacks
- Sound effects
- Menu screen
- Different characters
- Different stages
- Character skins
- Special effects
- Camera movement
- Blastzones
- Hitbox and hurtbox shifting
